# Яндекс Про Навигатор — бот-помощник для водителей

ИИ-бот в Telegram для водителей Яндекс Про в 12 странах.
Отвечает на вопросы о выплатах, документах, правилах и приложении,
используя **380 реальных статей** базы знаний Яндекс Про.

## Возможности

- 🌍 **12 регионов** — Россия, Казахстан, Беларусь, Узбекистан, Армения, Молдова, Грузия, Литва, Сербия, Кыргызстан, Таджикистан, Турция
- 💬 **Свободные запросы** — пишите вопрос своими словами
- 📂 **Категории + FAQ** — быстрые кнопки с проверенными запросами
- 🧠 **Гибридный поиск** — векторный + текстовый, 2467 чанков
- 🔗 **Реальные ссылки** — каждая статья ведёт на страницу базы знаний Яндекс Про
- 👍👎 **Обратная связь** — оценка качества ответа, кнопка «В меню»
- 📊 **Аналитика** — логирование запросов в SQLite

## Быстрый старт

### Docker (рекомендуется)

```bash
cp .env.example .env
nano .env  # BOT_TOKEN, YANDEX_API_KEY, YANDEX_FOLDER_ID
docker compose up -d --build
```

При первом запуске — автоиндексация ~4 мин (rate limiter, без 429).
Данные хранятся в Docker-томе `bot_data`.

### Локально

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env && nano .env
python -m rag.indexer
python -m bot.main
```

## Архитектура

```
Водитель → Telegram
  → Классификация (регион, категория)
  → Гибридный поиск (векторный + текстовый буст)
  → LLM-генерация (YandexGPT Lite)
  → Ответ + ссылка на реальную статью
  → Логирование (SQLite)
```

## Структура проекта

```
annya-mgmt/
├── bot/
│   ├── main.py            # точка входа, aiogram
│   ├── handlers.py        # RAG-пайплайн + fallback РФ
│   ├── keyboards.py       # меню: регионы, категории, FAQ
│   └── formatter.py       # Markdown, region_urls, экранирование
├── rag/
│   ├── indexer.py         # 380 статей → 2467 чанков → .npz
│   ├── retriever.py       # гибридный поиск (векторный + буст)
│   ├── classifier.py      # 12 регионов, 2 уровня ключевых слов
│   ├── llm.py             # YandexGPT API + rate limiter (9 req/s)
│   └── regions.py         # URL-маппинг 12 регионов
├── db/
│   └── analytics.py       # SQLite логирование
├── data/
│   ├── knowledge_base.json   # 380 статей (генерируется)
│   └── vector_store/         # .npz + metadata.json
├── data/raw_articles/        # исходные 549 JSON (scraped)
├── convert_kb.py             # конвертер 549 → 380 статей
├── generate_questions.py     # генератор 3762 вопросов
├── config.py
├── Dockerfile
├── docker-compose.yml
├── entrypoint.sh             # автоиндексация по хешу
└── requirements.txt
```
